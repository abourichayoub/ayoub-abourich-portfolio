### 📊 [PFE - Simulateur Python pour le Monitoring Énergétique (ISO 50001)](./PFE_Simulateur_Python_IIoT)
* **Organisme d'accueil :** Bank Al-Maghrib - Dar As-Sikkah.
* **Description :** Conception d'une architecture de monitoring énergétique non intrusive basée sur l'Internet des Objets Industriels (IIoT) pour optimiser l'efficacité de la ligne d'impression fiduciaire. 
* **Environnement & Outils :** Émulation de compteurs industriels via **Python**, transmission des données par **Modbus TCP/IP**, conteneurisation sous **Docker** et création de dashboards analytiques sur **Grafana**.
